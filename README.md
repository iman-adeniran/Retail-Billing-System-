PROJECT OVERVIEW
A desktop-based retail billing system built with Python and Tkinter. Designed to help small retail shops manage sales, calculate totals, and generate printable bills.

FEATURE
Add, remove, and update products
Automatic total and discount calculation
Print or save bills for customers
Clean, easy-to-use Tkinter interface

TECH STACK
Python
Tkinter

HOW TO RUN
Make sure you have Python installed.
Run the file:
python billing_system.py
Start adding items and generate your bill instantly.
